＜ 説明 ＞


今回、PythonでサッカーのPKゲームを制作することになりました。 ゲームは5回戦で、プレイヤーが3回以上ゴールできたら勝利、2回以下であれば敗退とします。 名前を記入してからゲームを開始してください。

＜ ソースコード ＞


  1. print("~ PKゲームにようこそ ~")
  2. name=input("プレイヤー名を入力してください。")
  3. print(name,"さんは、キッカーです。キーパーを出し抜いてシュートを決めよう！")
  4. point=0
  6. # 5回の試合を実行
  7. for i in range(1,6):
  8.      # プレイヤーがPKを蹴る
  9.  print(i,"戦目")
  10.  print("１．左上　２．左下　３．真ん中　４．右上　５．右下")
  11.  serect=int(input("蹴りたい場所の数字を入力してね"))
  13.  import random
  14.  a = random.randint(1,5)
  15.  print("キーパーが動いた方向は",a)
  16.  if serect==a:
  17.   print("止められた...。")
  18.  else:
  19.    print("ナイスシュートだー??")
  20.    point=point+1
  22.     # ゴール数を確認して勝敗を表示
  23. if point >= 3:
  24.         print(f"ゴール数: {point} - 点獲得で勝利！！")
  25. else:
  26.         print(f"ゴール数: {point} - 残念ながら敗退...。")
