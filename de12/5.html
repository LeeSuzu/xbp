  2. todo_list = []

  5. def add_todo(todo):

  10. def show_todos():
  11.     if todo_list:
  12.         print("ToDoリスト:")

  21.         deleted_todo = todo_list.pop(index - 1)
  22.         print(f"'{deleted_todo}' をToDoリストから削除しました！")
  23.     else:
  24.         print("無効なインデックスです。")

  42.         show_todos()

  44.             index = int(input("削除するToDoの番号を入力してください: "))

というPythonのソースコードが出てきました。

＜ 実際に使ってみた ＞

＜ 感想 ＞

Chat GPTを使用して見つけた【PythonによるTODOリスト】を作成するソースコードをもとに、
TODOリストの追加に加え、締切日ややる予定の日など日程も記入できるようにしたり、
予定の種類ごとにまとめて表示できるようにしたかったのですが、何度行っても反映されなかったり、
エラーが発生してしまったりと締め切り期限までに改良することができませんでした。
実用性があり、できたらとてもいい案だったのではないかと思いましたが、今回その考えを
実際に形にすることができなかったので、これからPythonやプログラミングについて学び、
思い描いたアイデアを形にできるように成長していきたいと感じました。
